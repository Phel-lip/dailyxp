# DailyXP

Aplicação gamificada de hábitos desenvolvida com React + Vite.

O projeto foi criado com foco em praticar fundamentos modernos de Front-End, componentização e gerenciamento de estado em React, mantendo uma arquitetura simples e explicável.

# DailyXP

Aplicação Full Stack de acompanhamento de hábitos gamificados.

## Tecnologias

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL

## Funcionalidades

- Criar hábitos
- Atualizar progresso
- Excluir hábitos
- Sistema de XP
- Sistema de Streak
- Recompensas
- Persistência em PostgreSQL

## Estrutura

/backend -> API .NET

/frontend -> Aplicação React
